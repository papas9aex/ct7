Отчет по проекту TaskManagerWithErrorHandling

Функциональность:
- Централизованная обработка исключений через AppDomain.CurrentDomain.UnhandledException.
- Логирование ошибок с деталями: сообщение, стек вызовов, уровень ошибки, контекст.
- Механизм оповещения через консоль: !!! ОШИБКА !!! (для привлечения внимания).
- Структурированный Trace лог для отладки.

Пример логирования ошибки:
!!! [Error] Ошибка в контексте: Add
Сообщение: Задача не может быть пустой.
Стек вызовов: ...

Применение:
- .NET Console Application
- System.Diagnostics.Trace
